# ISP Client App (Flutter)

Mobile version of the ISP dashboard with placeholder data and no backend.

## Features
- Overview dashboard with KPI cards, network health, and traffic snapshot
- Devices list with status and uptime
- Alerts list with severity indicators
- Model view with metrics and predictions

## Notes
- This is a front-end only build. Replace mock data in lib/main.dart with real API data when available.
- Designed for Android first, but the Flutter project supports other platforms if needed.
